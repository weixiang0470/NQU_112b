# **排序統計值濾波器**
- 中值濾波器、最大值濾波器、最小值濾波器
- 先排序周圍的值，然後選取目標位置的值，ex：中間值濾波器就會選中間值
- Median filter 可以得到水彩畫的效果

# **鈍化濾波器**
